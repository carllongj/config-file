### 备注
1. `ftplugin目录` 此目录是用以最先加载 `文件类型` 的配置,设置默认行为.
2. `after/ftplugin` 则是最后加载的配置,可以用于覆盖以及追加其它个性化设置.
3. 对于不同的语言对应的配置文件不是以文件后缀来决定的,而是根据其文件类型决定.
  * 通过 `:set filetype?` 来确定指定语言的类型是什么,以 `rust` 为例.
    * 文件后缀为 `rs`,但是语言类型为`rust`,因此定义 `rust` 的配置时
          使用`rust.lua`,而不是 `rs.lua`.
  * 文件名称必须要为如下格式,`<filetype>`表示文件后缀名称.
  ```
    <filetype>.lua
  ```

### 显示选项设置
* 带有 `LSP` 的语言可能不受全局配置显示选项如(`tabstop`)的控制,`LSP` 启动时可能
  会设置这些选项以符合其行为.
