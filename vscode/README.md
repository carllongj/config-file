### keybindings.json
* 该文件用以保存常用的快捷键设置,快捷键使用了部分`IDEA`的快捷键设定.
* 导入方式.
  1. 打开 `vscode` 命令面板,可通过如下快捷键打开.
    ```bash
      # 打开命令搜索面板
      ctrl + shift + p
    ```
  2. 打开键盘绑定快捷键设置,在命令面板中输入以下内容以检索到指定内容.
    ```bash
      # 键入搜索以下内容
      Preferences: Open Keyboard Shortcuts (JSON)
    ```
  3. 在新打开的`JSON`文件面板内容粘贴 [快捷键](./keybindings.json) 文件内容.
